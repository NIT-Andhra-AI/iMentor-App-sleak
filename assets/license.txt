2. PRIVACY & DE-IDENTIFIED TELEMETRY
-----------------------------------------------------------------------------
We collect ONLY de-identified, aggregated usage data to improve the product.

WHAT IS COLLECTED
  * A one-way cryptographic hash of your session identifier (the original UUID
    is never transmitted -- only its SHA-256 digest).
  * The mode you used: "general chat", "course mode", or "my documents".
  * Number of messages in a session and the time-to-first-token (milliseconds).
  * De-identified text of USER messages after the following PII is REMOVED before
    the data ever leaves your device:
        - Email addresses         ? replaced with [EMAIL]
        - Phone numbers           ? replaced with [PHONE]
        - Social security numbers ? replaced with [SSN]
        - URLs / web addresses    ? replaced with [URL]
        - Student / employee IDs  ? replaced with [ID]
        - Names with titles       ? replaced with [PERSON]
  * CPU core count and operating system name (e.g. "windows").

WHAT IS NEVER COLLECTED
  * The actual text of AI (assistant) responses -- only a SHA-256 fingerprint is
    stored so we can detect duplicate queries; the text itself is never sent.
  * Your name, username, email address, or any account credentials.
  * The name or hostname of your computer.
  * Your IP address (the server stores only a one-way hash, discarded after 24 h).
  * Any files you upload to the "My Documents" feature.
  * Passwords, tokens, or any authentication material.

WHEN DATA IS SENT
  Telemetry is transmitted ONLY when ALL of the following conditions are true:
    1. You have accepted this Agreement (your explicit consent is required).
    2. An active internet connection is detected.
    3. The app is in an IDLE state (no AI generation in progress).
  Sessions are queued locally and sent quietly in the background; they are
  never sent while you are actively using the AI.

HOW TO OPT OUT
  You may withdraw consent at any time:
    Settings ? Privacy ? "Send de-identified usage data" ? OFF
  Disabling this setting permanently prevents all future transmissions.
  Previously queued sessions that have not yet been sent are deleted from
  your device immediately.

DATA RETENTION
  De-identified session records are retained on the server for a maximum of
  12 months, after which they are permanently deleted.
