# C Programming Lab: Basic Patterns and Control Flow Programs

## Overview
This lab session provides hands-on coding exercises aligned to the **JNTUK R20 lab syllabus** for C Programming & Problem Solving. All programs use standard input/output as required by JNTUK practical examinations. Each program is presented with the problem statement, sample I/O, algorithm, and complete code.

## Lab Objectives
By the end of this lab, you should be able to:
1. Implement the core algorithms/data structures from scratch
2. Handle edge cases and validate input
3. Analyse time and space complexity of your implementation
4. Explain your approach in a viva voce examination

## Viva Questions (JNTUK Practical Exam)
1. What is the time complexity of your implementation?
2. How would you modify the program for a different input format?
3. What are the limitations of this approach?
4. Explain line X of your program.
5. What would happen if the input contains [edge case]?
