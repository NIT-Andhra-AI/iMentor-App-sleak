## How To Use This Course
1. Start with [Course Overview](overview.md) to understand the complete journey.
2. Follow Core Lectures in the order listed below (not alphabetical).
3. Move to Career Prep after finishing core concepts.

## Learning Path

### Getting Started
- [Design and Analysis of Algorithms — Wiki Index](index.md)
- [Design and Analysis of Algorithms — Course Overview](overview.md)

### Core Lectures (Syllabus Modules With Sequential Subtopics)

#### Module 1: Foundations of Analysis
Main lecture: [Asymptotic Analysis and Big-O Notation](asymptotic-analysis.md)
Then continue in sequence:
1. [Recurrence Relations and Master Theorem](recurrence-relations.md)
2. [Amortized Analysis: Aggregate, Accounting, Potential Methods](amortized-analysis.md)
3. [Computational Complexity Theory: The Architecture of Limits](complexity-theory.md)

#### Module 2: Divide-and-Conquer and Sorting Lineage
Main lecture: [Divide and Conquer](divide-conquer.md)
Then continue in sequence:
1. [Sorting Algorithms Analysis](sorting-algorithms.md)
2. [Linear-Time Sorting: Radix, Counting, and Bucket Sort](linear-sorting.md)

#### Module 3: Greedy and Dynamic Programming Design
Main lecture: [Greedy Algorithms: Theory, Design, and Engineering](greedy.md)
Then continue in sequence:
1. [Dynamic Programming: Principles and Practice](dynamic-programming.md)
2. [Advanced Dynamic Programming: Bitmask, Tree DP, Digit DP](dp-advanced.md)
3. [Approximation Algorithms for NP-Hard Problems](approximation-algorithms.md)

#### Module 4: Graph and Network Optimization
Main lecture: [Graph Algorithms: Foundations, Traversal, and Optimization](graph-algorithms.md)
Then continue in sequence:
1. [Minimum Spanning Trees: Prim and Kruskal](minimum-spanning-tree.md)
2. [Shortest Path Algorithms: Dijkstra, Bellman-Ford, Floyd-Warshall](shortest-path.md)
3. [Network Flow: Max-Flow Min-Cut, Ford-Fulkerson](network-flow.md)

#### Module 5: Strings and Search Spaces
Main lecture: [Advanced String Algorithms: From Pattern Matching to Suffix Structures](string-algorithms.md)
Then continue in sequence:
1. [Suffix Arrays, Suffix Trees, and Aho-Corasick](suffix-arrays.md)
2. [Backtracking: Algorithmic Search and Pruning](backtracking.md)

#### Module 6: Randomization and Specialized Domains
Main lecture: [Randomized Algorithms: Quickselect, Hashing, Monte Carlo](randomized-algorithms.md)
Then continue in sequence:
1. [Number Theory and Cryptographic Algorithms: GCD, Modular Exponentiation](number-theory-algorithms.md)
2. [Computational Geometry: Convex Hull, Line Sweep](computational-geometry.md)
3. [Online Algorithms and Competitive Analysis](online-algorithms.md)
4. [Parallel Algorithms: PRAM, Work-Span, MapReduce](parallel-algorithms.md)
5. [Competitive Programming Patterns and Templates](competitive-programming.md)
